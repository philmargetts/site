Resources

A (growing) list of some resources I have used and can recommend.

Education technology:

  • OneNote, Class Notebook, Learning Tools
  • PaperCut - print management

Communities:

  • Microsoft Tech Community
  • Microsoft Education Community
  • MITIE, ITMEL - IT in education
  • Spiceworks - IT Pro
  • USQ IT Students’ Network

Web/hosting:

  • CloudFlare - these guys make great stuff (and recently started offering wholesale cost domain hosting!)
  • iwantmyname - great registrar, lots of TLDs
  • GitHub Pages - Static web hosting, where this site lives
  • DigitalOcean - my referral link, where I have hosted a WordPress instance in the past
  • Discourse - clean yet powerful forums

Productivity:

  • TeXstudio - easy and powerful LaTeX document manipulation
  • Sumatra PDF - light-weight, super fast, love it
  • Atom - "hackable" text editor, dark theme by default!
  • paint.net - simple, powerful (for what I need) image editing
  • Notepad++ - text editor with syntax highlighting for multiple languages
  • Pencil - GUI prototyping
  • draw.io - powerful web-based diagramming (with desktop option)

Useful sites:

  • Bible websites: Bible Hub, BibleGateway, ESV.org
  • Wikipedia - think about it... virtually the sum of humanity's knowledge, accessible and editable by anyone. What an awesome resource!
  • DuckDuckGo - search engine that doesn't track you, including handy search click interactions and dark theme
  • txti - fast web pages for everybody. I appreciate their philosophy that fast internet is a responsibility
  • Tables Generator - graphical generator for LaTeX, HTML, Markdown syntaxes
  • Vim Cheat Sheet
  • Shell script quick reference

Android apps:

  • Fing
  • Wifi Analyzer
  • BOM - Australian Weather
  • Pong - just for fun
